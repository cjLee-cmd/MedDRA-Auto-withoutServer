## 8. 빠른 시작 가이드 (요약본)

사용자에게 제공할 1페이지 짜리 빠른 시작 가이드:

```markdown
# MedDRA 자동 코딩 시스템 - 빠른 시작

## 1단계: 로그인
https://cjlee-cmd.github.io/MedDRA-Auto-withoutServer/
- 사용자: acuzen
- 비밀번호: acuzen

## 2단계: PDF 업로드
- 왼쪽 "파일 선택" 버튼 클릭
- CIOMS-I 보고서 PDF 선택

## 3단계: 자동 검색
- 팝업에서 "자동 검색 시작" 클릭
- 검색 결과 확인

## 4단계: DB 자동 입력
- 왼쪽 "MedDRA-DB에 자동 입력" 버튼 클릭
- 새 탭에서 자동으로 데이터 입력됨
- "저장" 버튼 클릭

완료! 🎉

문제 발생 시: 사용자 매뉴얼의 "문제 해결" 섹션 참조
```
